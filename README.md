# AI-Powered Deep Learning Virtual Teacher (DLVT)

An interactive AI-powered Deep Learning education platform built using React, Vite, TailwindCSS, Framer Motion, Clerk Authentication, and Qwen AI APIs.

## Features

- Dynamic Deep Learning explanations
- Beginner / Intermediate / Advanced learning modes
- AI-generated roadmaps and quizzes
- Interactive neural network visualizations
- CNN, Transformer, GAN, RL, Diffusion Model support
- Secure Clerk authentication
- Responsive glassmorphism UI
- Real-time educational content generation

## Tech Stack

- React.js
- Vite
- TailwindCSS
- Framer Motion
- Clerk Authentication
- OpenRouter Qwen APIs
- React Flow
- Canvas API
- D3.js

## Modules

- Learn Topic
- AI Mentor
- Quiz Generator
- Visualization Engine
- Flashcards
- Study Planner
- Code Playground

## Installation

```bash
npm install
npm run dev
```
